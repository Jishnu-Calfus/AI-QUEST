Method

Each entity's history is split into an earlier baseline window and a recent window. Categorical features (action / tool / model mix) are compared with Jensen–Shannon divergence; numeric features with a standardized mean shift blended with % change. A blended score (0–1) grades stable → low → moderate → high. Read-only over the same event stream the live detectors use — this is the slow, distributional question they don't ask.
